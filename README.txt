Once you download the Zip file, please place the "MSGraph" file into C:\User\*USERPROFILE*\ directory. This will be the directory you will work from.

To run any of the commands, use when in PowerShell you will want to do for example - 

& "C:\Users\*USERPROFILE*\MSGraph\Intune\Android\DeleteDeviceUsingIMEI.ps1"

This will allow the scripts to run.
